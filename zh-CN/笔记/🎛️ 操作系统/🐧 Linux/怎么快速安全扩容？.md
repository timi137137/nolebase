# 怎么快速安全的扩容？

## Debian 系
对于Debian系列的系统来说扩容是非常简单的一件事。

`cfdisk`是一个命令行UI式的硬盘工具。
![[4f408a52-df4d-495b-9639-78c32730bdae.png]]
将需要扩容的分区后面的分区全删了，然后对需要扩容的分区`Resize`即可。

## Ubuntu 系
对于 Ubuntu系列的系统来说就复杂得多，运行以下命令[^1] ：
```bash
partprobe
pvresize /dev/sda3
lvextend -l +100%FREE /dev/mapper/ubuntu--vg-ubuntu--lv
resize2fs /dev/mapper/ubuntu--vg-ubuntu--lv
```

>[!warning] 
>请注意，以上路径并非固定，请依照不同系统自行替换，否则后果自负喵。
